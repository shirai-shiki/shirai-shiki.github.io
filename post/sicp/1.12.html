So logically, 5C2 for example is (4C1)+(4C2)

So we can assume nCr=(n-1)C(r-1)+(n-1)Cr for all n > 0 , r < n

So for n >= r, return 1, and for n < 0 return 0, otherwise recurse on the two upper values
